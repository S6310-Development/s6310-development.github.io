Building


Do know, this guide is made for people which already have a good understanding of how Linux works. I will not help people who need help with every single little thing they do. If you do not understand what you are doing then go back to writing VB scripts or whatever.

Step 1


Run Linux.. This is required to build Android. No explanation needed.

Step 2


Install all the required packages. These vary from Android version, it's best to simply Google this.

Step 3


Create a folder in your Home folder named 'android' and another folder in your Home folder named 'bin'.

Step 4


Open a Terminal and run 'curl http://commondatastorage.googleapis.com/git-repo-downloads/repo > ~/bin/repo'. Next run 'chmod a+x ~/bin/repo'.

Step 5


Now open '~/.bashrc' with a text editor (you may require to show hidden files) and add the line 'export PATH=${PATH}:~/bin' to the top of the file. OPTIONAL: Add the path to your adb and fastboot files too.

Step 6


Open up a Terminal again and type 'cd ~/android'. Next run 'repo init -u git://github.com/CyanogenMod/android.git -b cm-11.0' and then run 'repo sync'. This may take a long time depending on Internet speed.

Step 7


Get all the device specific code at https://github.com/S6310-Development. Be sure to delete the folders you are replacing first or you may get errors. NOTE: In 'external/bluetooth' there is a folder named 'bluedroid', this one has to be removed no matter what.

Step 8


Create a build script to automate the process of building, cleaning and so on. Happy building!
